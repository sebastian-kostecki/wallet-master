## Global standards (apply in every task)

- [ ] **Invalid fields:** `aria-invalid="true"` on input when `error` present; `InputError` has `role="alert"` and `:id="forId + '-error'"`; input `aria-describedby` points to error id.
- [ ] **Loading:** submit buttons `:disabled="form.processing"` + `aria-busy="true"` when processing; visible spinner already present — add `aria-label` from i18n.
- [ ] **Dialogs:** Radix/shadcn `Dialog` — verify `aria-labelledby` / `aria-describedby` via `DialogTitle` + `DialogDescription`; Esc closes; focus returns to trigger.
- [ ] **Focus:** global rule in `app.css`:

```css
:focus-visible {
    outline: 2px solid hsl(var(--ring));
    outline-offset: 2px;
}
```

- [ ] **Tables:** `transactions/Index` — `<table>` with `<caption class="sr-only">` or `aria-label` on table; sort buttons `aria-sort="ascending|descending|none"`.
- [ ] **i18n:** Auth pages still in English in template — migrate visible strings to `$t()` / `pl.json` where product language is Polish (PRD §7).
